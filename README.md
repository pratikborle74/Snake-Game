# 🐍 Enhanced Snake Game  

A **classic snake game** built using Python's `turtle` module, featuring **smooth movement, screen wrapping, food animations, and an engaging UI**.

---

## 🎮 Features  
✅ **Smooth Movement** with directional key controls  
✅ **Food System** with animations when eaten  
✅ **Screen Wrapping** (snake teleports to the opposite side when crossing boundaries)  
✅ **Game Over Handling** (resets after collision)  
✅ **Custom Background Image Support** (`bg.gif`)

## 🚀 Installation & Usage
### **1️⃣ Clone the Repository**
```sh
git clone https://github.com/pratikborle74/Enhanced-Snake-Game.git
cd Enhanced-Snake-Game
```

### **2️⃣ Install Dependencies**
```sh
pip install -r requirements.txt
```

### **3️⃣ Run the Game**
```sh
python src/snake_game.py
```

---

## 🎮 Controls  
🔼 **Arrow Up** → Move Up  
🔽 **Arrow Down** → Move Down  
◀️ **Arrow Left** → Move Left  
▶️ **Arrow Right** → Move Right  

---

## 📜 Requirements
- **Python 3.x**
- **turtle** (built-in)
- **random** (built-in)
- **time** (built-in)

Ensure you have Python installed before running the game.
